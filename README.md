# PolyFlow-Backend

This is the Backend Part of the Project

PolyFlow is a Hobby Project that aims to provides multiple utilities to the User. for eg. ToDoTask task management and many more utilities which will be added in the project with time...

This is a gradle based project that was generated with [spring-initializr](https://start.spring.io/) with Spring boot version 2.3.8 and Java version 8.

## Development server

Run `gradle clean build` and then `gradle bootRun` for a dev server. Navigate to `http://localhost:${port}/`. The server port can be configured from application.yml file
